用小米盒子3增强版看 YouTube 与 Netflix 教程

发表于 2017-08-23 | 分类于 教程 | | 阅读次数
不需要root，让你实现利用小米盒子在电视观看 Youtube 和 Netflix

有些日子没有更新了，博主最近和小伙伴们一起搞了点事情买了一款壕级的 HK 线路，然后博主利用搞到的 HK 线路看上了 Netflix， 在用 PC 看的时候始终觉得不够爽，屏幕太小，于是就想到了是不是可以直接在家里的小米盒子上继续搞一把事情呢？有想法那就开始行动。

工具准备：小米盒子3增强版、能搭载 shadowsocks 服务的路由器、一条能看 Netflix 的线路、安卓手机（安装好投屏神器）

一、路由器

博主使用的是Netgear 4300 （JD购买链接）

这款机器有 V1、V2、V3 等版本，一定要注意只有 V1 版本才有民间编译的 openwrt 固件，刷机后能支持 shadowsocks 服务以及其他强大服务，这样就实现了路由器端的网络破墙，可以让你手机、平板、电脑、电视盒子实现直接访问墙外网络。
下面提供刷机固件和刷机教程的链接，请有需求的朋友自己研究一下，不难。

石像鬼固件地址

固件使用及刷机教程

购买该路由器前务必询问清楚机器版本，JD 没有 V1 的话可能只有天猫或者淘宝了，这款路由器价格从三年前的 259 到一年前的 299 到现在的 359，价格也是持续走高。
如果觉得 4300 这款太贵，还可以选用华硕的 AC 系列中支持刷梅林固件的型号，博主没有使用经验不赘述具体可以自行搜索。

二、网络线路

众所周知 Netflix 从 2015 年开始就锁区了，并且有强大的代理监测手段，使得大家想通过购买 VPS 搭建 shadowsocks 服务来观看 Netflix 这种方式变得无法实现。


利用上一篇教程自己搭建的 GCP 的线路访问观看 Netflix 就是上图这种效果。

为了丰富业余生活，想看 Netflix 的心还是不死，那么我们找到了一家靠谱的 服务商（点击前往），付费购买服务实现解决网络线路的问题，为了避免广告嫌疑，也不赘述，各位可以访问后自行判断。
当然如果你不看 Netflix， 仅仅想通过电视看 YouTube， 那我们之前搭建的 GCP 的线路也是够用的。

三、需要在小米盒子上完成的工作

前不久刚把用了三年的小米盒子1s淘汰掉，购入了小米盒子3增强版，以下内容 都是只针对小米盒子3增强版这个平台，其余硬件的情况一概不清楚，有兴趣的朋友可以自行摸索，大同小异。

完成了步骤一和步骤二，意味着我们的小米盒子3增强版可以直接访问 YouTube 和 Netflix， 那么我们只需要给盒子安装上合适的APK文件，就可以实现利用盒子观看 YouTube 与 Netflix 了。

点击进入搜索链接


点击该链接进入选择 arm64 版本的apk文件进行下载

3.安装APK文件

文初在准备事宜里有提到一部安装了 投屏神器 的安卓手机，你可以用这台安卓手机下载好这两个 APK 文件，然后利用 投屏神器 直接安装到盒子上；当然也可以用其他你常用的方式来实现安装。安装完毕后，你小米盒子里就多了这两个程序。

4.程序的使用

如果只使用遥控器的话，Netflix的APP对小米系统的支持不是太完美，这时候你可以用自己的无线鼠标和键盘套装，把发射器插在盒子背后的USB接口，就可以在小米盒子上实现用鼠标操作以及键盘输入登录。


鼠标键盘完美融入使用

针对 YouTube ，由于没有 Root ，所以无法安装谷歌全家桶，所以无法登陆谷歌账户，但是，一样是有解决方案的。
先在电视上进入 YouTube 后用遥控器进入左侧的设置界面，点击进入 关联电视及手机，选择通过电视代码关联

同时打开你手机的 YouTube APP，进入设置里下面这个界面，把你在电视上看到的数字输入进手机里，就连接上了。

这样就可以通过手机上选择播放的内容，然后在电视上直接播放，而免去了电视登录 YouTube 需要root设备这一麻烦的程序。值得一提的是，YouTube 这种把电视和手机连接起来的方式跟大家熟知的投屏是不一样的，投屏是手机播放然后通过局域网串流到电视，而 YouTube 这种方式流量是直接走的小米盒子的客户端并不需要走手机客户端，手机这边只相当于一个选择遥控器。
